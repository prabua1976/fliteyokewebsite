Services Explained

Experience Flying on a Simulator

Our Cessna 172 Glass Cockpit Simulator is available for students and other enthusiasts to experience the joy of flying. This will be on slots of one hour each.

One Day Program for Young Minds

For young minds, we also provide an one-day overview program. This program covers the basics of flying, aircraft structure, forces acting on an aircraft, various controls, flight instruments etc. Young minds get to learn a lot about the science behind flight in a hands-on and practical approach. This program includes simulator time. Participants will be provided with a courseware and a program certificate.

PRACTICAL DEMONSTRATIONS:

* Foam Model to explain aerodynamic structure of an airplane

* RC Model for demonstrating how ailerons, rudders and elevators work

* RC Brushless Motor with props to demonstrate THRUST

* RC Airplane Electronics - Transmitter, Receiver, Servos, Brushless Motor

* Cessna 172 Glass Cockpit Flight Simulator for flying experience

TYPICAL SCHEDULE:

10:00AM - 10:30AM Induction and courseware distribution

10:30AM - 11:30AM Overview / Basics

11:30AM - 12:00PM Demo I using Foam Model

12:00PM - 12:30PM Demo II using working Model Airplane

12:30PM - 01:30PM Lunch Break

01:30PM - 02:00PM Demo III - RC Airplane Electronics

02:00PM - 02:30PM Demo IV - Flight Simulator

02:30PM - 05:00PM Flight Simulator Practice Session

05:00PM - 05:30PM Q & A and Wrap-up
